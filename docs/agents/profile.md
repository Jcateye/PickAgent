# Agent Profile

Selected profile: `coding`.

Use this profile for software development, debugging, release work, architecture review, and technical documentation.

## Defaults

- Prefer repo-local evidence over assumptions.
- Keep changes scoped to the task.
- Use `software-assets-builder` when initializing projects, loading business modules, generating schema/admin assets, or validating generated scaffolds.
- Use tests, builds, lint, curl, browser checks, or app smoke tests as verification.
- Track risk as `L0` to `L3`.
- Ask before dependency upgrades, migrations, auth/payment changes, deployment changes, or module-boundary changes.
